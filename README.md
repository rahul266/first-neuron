# First Neuron

This is very atomic implementation of Neural network and How Backpropogation, autograd happens during training the neural network.

`Class Value` is the atomic level in calculation if any 2 entities. This Value class has a method which back propogates and calculate the gradiant based on the operation perrformed on the value.

This Project is a learning step in neural networks from micrograd repo hence it resembles Andrej Karpathy's micrograd.
